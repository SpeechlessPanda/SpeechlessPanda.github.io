项目收藏

发表于2026-06-06|更新于2026-06-24
|总字数:3.7k|阅读时长:12分钟|浏览量:

这篇博客是由 AI 编写，但里面的项目是由我选取。后续会持续更新。

收集 GitHub 上值得关注的优质开源项目，按类别整理，持续更新。Star 数截至 2026-06-15。🆕 标记表示最近一次同步新增的项目，点击下方分类标题可展开查看详情。

AI 编码代理

这类项目是当前最火的赛道——让 AI 直接在终端里帮你写代码、跑任务。

  • Cherry Studio ⭐ 47.4k — AI 生产力工作室，集成智能对话、自主代理和 300+ 助手，统一接入前沿 LLM。桌面 GUI 应用，适合非终端用户高效使用 AI。亮点是开箱即用的丰富助手生态和跨平台桌面体验。

  • Goose ⭐ 49.5k — 开源可扩展 AI 代理。超越简单的代码建议，支持安装、执行、编辑和测试，可搭配任意 LLM 使用。亮点在于高度可扩展的插件架构，面向终端和自动化场景。

  • Warp ⭐ 61.8k — 诞生于终端的智能开发环境。将传统终端升级为 AI 驱动的 agentic 开发环境，集成了智能补全、命令搜索和工作流自动化。

  • Toad ⭐ 3.2k — 终端 AI 统一接口。用一个工具统一管理不同的 AI 服务，解决终端环境下多 AI 后端切换的痛点。

  • CC-Switch ⭐ 101.5k — 跨平台桌面 All-in-One 助手。统一管理 Claude Code、Codex、OpenCode、OpenClaw、Gemini CLI 等多种 AI 编码工具，一站式切换。亮点是真正做到了多工具统一管理。

  • Codex Web ⭐ 150 — Codex Desktop 的浏览器前端。让你通过网页界面远程控制本机的 Codex，解决了纯终端交互不够直观的问题。

  • Pi ⭐ 62.8k — AI 代理工具包，包含交互式编码代理 CLI、统一多提供商 LLM API（支持 OpenAI、Anthropic、Google 等）、Agent 运行时、TUI 终端 UI 库和 Slack 机器人。亮点是单体仓库架构，从底层 LLM 调用到上层编码代理全链路自研，供应链安全做得非常严格（依赖锁定、签名审计、shrinkwrap）。

AI 代理技能与框架

这类项目关注 AI 代理的能力扩展和编排方式。

  • Superpowers ⭐ 228.5k — 代理技能框架与软件开发方法论。提供一套完整的 agentic skills 框架，让 AI 编码工具遵循结构化的开发流程。亮点是经过实战验证的方法论，而非理论框架。

  • 🆕 Matt Pocock Skills ⭐ 129.6k — 面向”真实工程”的实战代理技能集。来自 Total TypeScript 作者 Matt Pocock 的 .claude 目录，提供 /grill-me、/grill-with-docs 等需求对齐技能，解决代理与用户需求错位的核心痛点。亮点是技能小巧、可组合、易改造，适配任意模型。

  • OpenCLI ⭐ 24.4k — 将任意网站变成 CLI 工具。借助 AI 代理使用你已登录的浏览器，把 Web 操作转化为命令行交互。亮点是支持已登录状态的浏览器复用，绕过认证难题。

  • MemPalace ⭐ 55.7k — 开源 AI 记忆系统。目前基准测试表现最好的开源 AI 记忆方案，且完全免费。解决了 AI 代理缺乏长期记忆的核心问题。

  • Herdr ⭐ 5.7k — 终端代理多路复用器。像一个”代理管理器”，可以同时调度多个 AI 代理协同工作，适合复杂的多步骤任务场景。

  • Awesome Persona Distill Skills ⭐ 4.4k — 以人物关系、纪念场景和方法论视角为主题的代理技能集合。解决的是 AI 在人际交往和情感场景下的表达能力问题。

AI 设计与前端

这类项目用 AI 提升设计质量和前端开发效率。

  • Open Design ⭐ 65.2k — 本地优先的开源 Claude Design 替代品。原生桌面应用，内置 259+ 技能和 142+ 设计系统，支持 Web/桌面/移动端原型、幻灯片、图片、视频，可导出 HTML/PDF/PPTX/MP4。亮点是设计系统生态极其丰富，适配几乎所有主流 AI 编码工具。

  • Impeccable ⭐ 38.6k — 让 AI 更擅长设计的设计语言。提供一套设计规范，显著提升 AI 生成 UI 的视觉质量，解决 AI 输出”看起来像 AI 做的”这个普遍问题。

  • Awesome Design MD ⭐ 90.4k — 知名品牌设计系统的 DESIGN.md 文件合集。将设计系统以 AI 可读的方式呈现，让编码代理能生成符合品牌规范的 UI。

  • UI/UX Pro Max Skill ⭐ 91.9k — 为 AI 提供专业级 UI/UX 设计智能的技能包。覆盖多平台设计规范，让 AI 在生成界面时具备专业设计师的判断力。

  • Taste Skill ⭐ 44.2k — 给 AI 注入”好品味”。阻止 AI 生成无聊、通用的设计输出，让结果更具个性和辨识度。

  • Frontend Slides ⭐ 21.7k — 利用编码代理的前端技能在 Web 上创建精美幻灯片。巧妙地将 AI 的前端编码能力转化为演示文稿制作能力。

  • Open Slide ⭐ 5.3k — 专为 AI 代理打造的幻灯片框架。将幻灯片制作抽象为代理可执行的标准化流程，适合自动化生成演示内容。

  • Ian Xiaohei Illustrations ⭐ 4.7k — 中文小黑怪诞风格正文配图生成 Skill。16:9 白底手绘风格，搭配少量红橙蓝批注色彩，专为文章配图场景设计。亮点是将独特的怪诞美学风格标准化为可复用的 AI 技能，一键生成风格一致的插图。

AI 提效与优化

这类项目关注如何让 AI 工具跑得更快、用得更省。

  • RTK ⭐ 62.5k — CLI 代理，在常见开发命令的输出传入 LLM 之前进行过滤和压缩，减少 60-90% 的 token 消耗。单个 Rust 二进制文件，零依赖，支持 100+ 命令，开销 <10ms。亮点是极低性能开销和零侵入接入（只需设为代理即可生效）。

  • Headroom ⭐ 28.3k — 在工具输出、日志和 RAG 数据传入 LLM 之前进行压缩，减少 60-95% 的 token 消耗且不影响回答质量。提供库、代理和 MCP Server 三种接入方式。亮点是显著的 token 成本节省。

  • Readme AI ⭐ 2.9k — AI 驱动的 README 文件生成器。自动分析项目结构和代码，生成专业规范的 README 文档，省去手动编写的繁琐工作。

学术科研工具

面向研究人员的效率工具，覆盖论文写作、绘图、灵感获取等环节。

  • Paper2Poster ⭐ 3.8k — NeurIPS 2025 发表的多代理论文海报生成系统。输入一篇论文，自动输出专业级学术海报。亮点是多代理协作架构，质量远超模板填充方案。

  • PaperX ⭐ 68 — 论文写作辅助工具。帮助研究人员提升论文写作效率，专注于解决学术写作中的痛点问题。

  • Nature Skills ⭐ 20.2k — 让 AI 具备 Nature 级别的学术表达能力。包含符合顶级期刊要求的学术写作风格和科研绘图技能，解决 AI 写论文”不够学术”的问题。

  • Idea Stream ⭐ 5 — 按主题从 arXiv 获取研究灵感，自动按质量、相关性和时效性排序。帮助研究者高效跟踪前沿进展，发现研究空白。

  • Diagrams ⭐ 625 — 物理学、化学和机器学习核心概念的精美示意图集。使用 Typst 编写，适合直接用于论文、教学和学术演示。

  • Touying Exporter ⭐ 131 — 为 Touying 演示框架提供多格式导出支持。解决 Typst/Touying 生态中导出格式受限的问题。

排版模板

各种好用的排版模板。

  • Universal HIT Thesis ⭐ 60 — 哈尔滨工业大学 Typst 论文模板。覆盖本科到博士各学位类型的论文排版需求，用 Typst 替代 LaTeX 实现更快的编译速度和更简洁的语法。

  • Quite Elegant Typ ⭐ 21 — 基于 Typst 的 ElegantBook 复刻模板。将 LaTeX 生态中广受欢迎的 ElegantBook 书籍风格移植到 Typst，适合制作精美笔记和技术文档。

学习资源与教程

优质的编程学习资源和教程合集。

  • Hello 算法 ⭐ 126.8k — 动画图解、一键运行的数据结构与算法教程。支持十余种编程语言实现，以可视化方式降低算法学习门槛。亮点是动画演示质量极高。

  • LeetCode Book ⭐ 8.3k — 《剑指 Offer》和《图解算法数据结构》配套代码合集。提供 Python、Java、C++ 三种语言实现，是面试刷题的经典参考。

  • Build Your Own X ⭐ 515.8k — 通过重建你喜爱的技术来掌握编程。涵盖从数据库到操作系统到编译器的各种”造轮子”教程，是深入理解计算机系统原理的最佳实践路径。

  • Project Based Learning ⭐ 269.5k — 项目驱动学习教程合集。按编程语言分类，每个项目都有明确的学习目标和完整的实现步骤。

  • Learn Claude Code ⭐ 66.6k — 从零构建类 Claude Code 的代理框架。以 “Bash is all you need” 为理念，手把手教你搭建一个 nano 级 agent harness，适合理解 agentic coding 的底层逻辑。

  • 计算机自学指南 ⭐ 73.2k — 北京大学同学整理的计算机科学自学路径。涵盖从入门到进阶的完整课程体系，是 CS 自学者的路线图。

  • TorchCode ⭐ 4.2k — PyTorch 版 LeetCode。通过从零实现 softmax、注意力机制、GPT-2 等核心组件来深度学习深度学习，自带自动评分。亮点是 Jupyter 交互式学习体验。

  • MiniMind ⭐ 51.8k — 2 小时从零训练一个 64M 参数的大语言模型。将 LLM 训练的完整流程浓缩到最小规模，是理解大模型原理的最佳入门项目。

  • Awesome OpenSource AI ⭐ 3.9k — 精选真正开源的 AI 项目、模型、工具和基础设施合集。帮助快速了解开源 AI 生态全貌。

实用工具

解决日常开发和工作中的实际需求。

  • LinkSwift ⭐ 16.5k — 八大网盘下载地址获取工具。支持百度网盘、阿里云盘、中国移动云盘、天翼云盘、迅雷云盘、夸克网盘、UC 网盘、123 云盘，一键获取直链。

  • Lark CLI ⭐ 14.2k — 飞书/Lark 官方命令行工具。覆盖消息、文档、多维表格、日历、邮件等核心业务场景，提供 200+ 命令和 20+ AI 代理技能，是飞书深度用户的效率神器。

  • InputTip ⭐ 3.2k — 输入法状态管理工具。实时提示当前输入法状态（中/英），并支持快速切换。解决 Windows 下频繁误输入中英文的问题。

  • FRP Panel ⭐ 1.7k — 多节点 FRP 内网穿透管理面板。提供可视化 Web UI 管理 FRP 服务端和客户端，可作为 Cloudflare Tunnel、Tailscale Funnel 等方案的开源替代。

  • Public APIs ⭐ 441.7k — 免费 API 合集。涵盖几乎所有类别的公开 API 列表，是开发者寻找第三方服务接入点的首选参考。

  • Career Ops ⭐ 53.8k — 基于 Claude Code 的 AI 求职系统。提供 14 种技能模式、Go 驱动的仪表盘、PDF 生成和批量处理，覆盖简历优化到面试准备的全流程。亮点是将 AI 代理能力系统性地应用于求职场景。

  • MapPoster Online ⭐ 667 — 把你爱的城市变成精美海报。浏览器端地图海报生成器，无需安装即可将任意城市生成极简风格的地图艺术海报，适合做家居装饰或独特礼物。亮点是基于 maptoposter Python CLI 的网页升级版，开箱即用。

  • Termux ⭐ 56.4k — Android 终端模拟器和 Linux 环境。在手机上获得完整的 Linux 命令行体验，通过 apt 包管理安装 Python、Node.js、Git 等开发工具链。亮点是无需 root 即可运行，让移动设备瞬间变成便携开发环境。

安全研究
  • PoC Lab ⭐ 617 — 最新 CVE 漏洞的 PoC 和复现脚本合集。聚焦 Linux 内核、Windows、macOS 等平台的高危漏洞，适合安全研究人员跟踪最新漏洞动态和进行漏洞复现实验。
创业与营销
  • Marketing for Founders ⭐ 6.4k — 面向独立开发者和创业者的实战营销资源。从获取前 10 个用户到 1000 个用户，提供分阶段的可执行策略，适合技术背景的创始人补齐营销短板。
同类项目对比

以下是功能相近的项目横向对比，方便按场景快速选择。

AI 代理平台：Cherry Studio vs Goose

维度 Cherry Studio Goose
交互形式 桌面 GUI 应用 终端 CLI
定位 通用 AI 生产力工具 开发者自动化代理
LLM 支持 统一接入多家前沿模型 支持任意 LLM
扩展方式 300+ 内置助手市场 插件架构，可自行开发
适合人群 非技术用户、产品经理、内容创作者 开发者、DevOps、自动化场景
选择建议 想要图形界面、开箱即用 → Cherry Studio 想要终端集成、自动化流程 → Goose

Token 压缩：RTK vs Headroom

维度 RTK Headroom
实现语言 Rust（单二进制，零依赖） TypeScript
压缩方式 CLI 代理，过滤和压缩命令输出 库/代理/MCP Server，压缩日志和 RAG 数据
支持范围 100+ 常见开发命令（git、ls、cat 等） 通用文本压缩，不限命令类型
性能开销 <10ms 较低
接入方式 设为系统代理即可生效，零配置 需选择库/代理/MCP 其中一种接入
压缩率 60-90% 60-95%
选择建议 追求极致性能和零配置 → RTK 需要灵活接入方式或压缩 RAG 数据 → Headroom

工程代理技能：Superpowers vs Matt Pocock Skills

维度 Superpowers Matt Pocock Skills
作者背景 软件开发方法论实践者 Total TypeScript 作者（TS 教育权威）
设计哲学 完整方法论，端到端流程 小巧、可组合、易改造
核心能力 TDD、调试、计划等结构化开发纪律 /grill-me、/grill-with-docs 需求对齐
解决痛点 代理缺乏工程纪律和方法 代理与用户需求错位、输出冗长
模型适配 主要面向 Claude Code 适配任意编码代理和模型
扩展方式 框架化，按 skill 规范加载 可直接 fork 改造成自己的技能
适合人群 想要完整工程方法论的团队 想要轻量灵活技能的个人开发者
选择建议 想要系统化、有纪律的开发流程 → Superpowers 想要轻量、即拿即改的实用技能 → Matt Pocock Skills
